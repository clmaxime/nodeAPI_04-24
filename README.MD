# Node API

Ce projet en NodeJS est un catalogue de vêtements avec des réductions.

## Dépendances :

- Express
- Express-validator
- Dotenv
- Jsonwebtoken
- Mongoose
- Multer
- Body-parser

## Dépendances Dev :
- @babel/core
- @babel/node
- @babel/preset-env
- @types/jest
- Jest
- Nodemon
- Supertest

## Installation du projet :
##### Cloner le projet
```
git clone
```
##### Alelr dans le répertoire du projet
```
cd projet
```
##### Installer les dépendances
```
npm install
```
##### Lancer le projet
```
npm run start:dev
```
